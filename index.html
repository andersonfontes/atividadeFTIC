Orçamento Técnico

Realizar uma pesquisa de peças para montagem de um computador para a seguinte aplicação: programaçlão front-end.

O computador deverá ter os seguintes componentes:

  1. PROCESSADOR
  2. PLACA-MÃE
  3. PLACA DE VIDEO
  4. MEMÓRIA RAM
  5. FONTE ATX
  6. GABINETE
  7. MONITOR
  8. TECLADO
  9. MOUSE
  10. HD / SSD
  11. COOLER + DISSIPADOR

O orçamento disponibilizado é de R$2.500

Realizar pesquisa em dupla, colocar os itens nesta pagina organizados da forma que você achar melhor utilizando as tags html que voce já conhece

Ao final, efetuar git add, git commit e git push e verificar se a pagina está publicada
